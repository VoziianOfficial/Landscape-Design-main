# Verdeon Image Sources

## Production photography

All landscape photographs used by this project were generated specifically for Verdeon on July 20, 2026 with OpenAI's built-in image generation tool. They do not copy the supplied screenshots, template branding, demo text, testimonials, or named projects.

The generated originals remain in the local Codex generated-image store. Project-consumed versions were cropped where indicated and encoded locally with Google's official libwebp 1.6.0 `cwebp` utility at quality 84.

## Prompt direction

The final prompt set requested premium editorial landscape-architecture photography with believable horticulture, natural materials, restrained color, architectural composition, and no logos, watermarks, direct-contractor messaging, oversaturated greens, or template-like stock staging. Service diptychs explicitly requested independent scenes for residential, commercial, front-yard, backyard, planting, patio/walkway, lighting, and visualization use. The planning prompt requested a candid plan review and a separate top-down 2D plan. The gallery prompt requested four independent concept studies in a precise 2×2 grid.

## Brand artwork

The Verdeon two-leaf logo, favicon SVG, and topographic line pattern are original project-native SVG artwork. They were created directly as vector code and do not use stock icons or generated raster logo imagery.
